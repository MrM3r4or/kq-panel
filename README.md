# 🤖 kQ Panel v1.5 – پنل مدیریت ربات تلگرام

پنل مدیریت حرفه‌ای ربات تلگرام با پشتیبانی از **چند ربات هم‌زمان**، **دستیار هوش مصنوعی**، **آپلود فایل**، **جوین اجباری**، **پیام همگانی** و **بخش‌های اختصاصی برای هر نوع ربات**.

---

## ✨ امکانات

- 📁 **آپلود فایل** و دریافت لینک اختصاصی (`t.me/BotUsername?start=XXXX`)
- 🧠 **دستیار هوش مصنوعی** با قابلیت تنظیم پرامپت، کلمات ممنوعه، مزاحم نشوید (DND)
- 👥 **مدیریت کاربران**
- 💬 **تنظیم پیام‌های سفارشی**
- 🔘 **ساخت دکمه‌های شیشه‌ای و منویی**
- 🔗 **جوین اجباری** (چند کانال)
- 🚫 **مدیریت بن کاربران**
- 🌙 **تم دارک و لایت**
- 📊 **لاگ لحظه‌ای**
- 📢 **پیام همگانی** با زمان‌بندی
- ⚙️ **تنظیمات** (تغییر توکن، مسیر ذخیره‌سازی، خروج)

## 🛠️ نحوه نصب و راه‌اندازی

### پیش‌نیازها
- Python 3.8+
- pip

### مراحل نصب
```bash
# ۱. فایل زیپ را دانلود و اکسترکت کنید
# ۲. کتابخانه‌های مورد نیاز را نصب کنید
pip install -r requirements.txt

# ۳. برنامه را اجرا کنید
python app.py

# ۴. مرورگر را باز کرده و به آدرس زیر بروید
http://127.0.0.1:5000
