پنل مدیریت حرفه‌ای ربات تلگرام با قابلیت آپلود فایل و مدیریت کامل

## ✨ امکانات

- 📁 آپلود فایل و دریافت لینک اختصاصی
- 👥 مدیریت کاربران
- 💬 تنظیم پیام‌های سفارشی
- 🔘 ساخت دکمه‌های شیشه‌ای و منویی
- 🔗 جوین اجباری
- 🚫 مدیریت بن کاربران
- 🌙 تم دارک و لایت
- 📊 لاگ لحظه‌ای
